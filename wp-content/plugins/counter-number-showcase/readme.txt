Counter Number plugin design layout is based on latest css animation and support on all major browsers. Easily upload Counter Number statistics. Easily update their records like title, counter stats, and icons.

== Installation ==

1. Upload the entire `counter-number-showcase` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'counter number' menu in WordPress.
3. Go to teams to customize this plugin.
4. Finally Enjoy it.
